# 더비 푸시 알림 템플릿 Top 30 v1

목적: 사용자가 잠금화면/알림센터에서 보고 “뭐야 이색기…?” 하고 눌러보게 만드는 짧은 푸시 알림.

톤 기준:
- 더비는 웃기려고 하지 않는다. 그냥 진지하게 이상한 보고를 한다.
- 핵심은 “멍청한데 당당함”과 “예측 안 되는 엉뚱함”이다.
- 짧고 간결해야 한다. 설명이 길어지면 맛이 빠진다.
- 제목은 평범하거나 진지하게, 본문은 얼탱이 없게.
- 액션 버튼 없음. 클릭하면 앱의 오늘의 사고/업무/일기장 화면으로 이동.

권장 운영:
- 하루 최대 3회.
- 같은 문구는 최소 30일 재노출 금지.
- 너무 비슷한 계열의 문구는 하루에 1개만.
- BM/결제 유도 문구는 별도 카테고리로 분리하고, 기본 푸시에서는 과하게 쓰지 않는다.

## 후속 확장 규칙

새 푸시를 추가할 때는 아래 기준을 통과해야 한다.

1. 더비가 본인 기준으로는 진지한가?
2. 문장이 짧은가?
3. 설명 없이도 얼탱이가 없는가?
4. 사람이 친 농담처럼 보이지 않는가?
5. “멍청한데 당당함” 또는 “예측 안 되는 엉뚱함” 중 하나가 살아있는가?
