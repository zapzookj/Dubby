# 더비의 오늘의 업무 템플릿 Top 30 v2

## 문서 목적

이 문서는 더비 앱의 홈 화면 또는 `오늘의 업무 보고` 화면에 표시할 고품질 업무 템플릿 30개를 정의한다.

푸시 알림은 앱 진입을 유도하는 짧은 어그로 문구지만, 오늘의 업무는 앱에 들어온 사용자에게 주는 보상 콘텐츠다. 따라서 푸시보다 길고, 더비가 실제로 무언가를 수행한 척하는 보고 형식을 유지한다.

## 톤 기준

- 더비는 웃기려고 하지 않는다.
- 더비는 무능하지만 당당하다.
- 더비는 형식적 성공에 집착한다.
- 더비는 결과가 이상해도 성과로 보고한다.
- 문장은 짧게 유지하되, 푸시보다 한 단계 더 상황성이 있어야 한다.
- 너무 영리한 말장난, 풍자, 직장인 농담, 철학 개그는 피한다.

## 사용 방식

하루에 3개씩 랜덤 또는 가중치 기반으로 노출한다.
같은 사용자에게 동일 업무가 반복 노출되지 않도록 `shown_at` 이력을 저장한다.

추천 UI:

```text
오늘의 업무 보고

✅ 업무명
결론: ...
비고: ...
```

`비고`는 선택 사항이다. 화면이 좁다면 업무명과 결론만 표시해도 된다.

## 운영 메모

오늘의 업무는 푸시보다 더 긴 호흡의 개그 콘텐츠다.
따라서 푸시처럼 한 문장으로 끝내기보다, “더비가 무언가를 수행한 척했다 → 결과가 이상하다 → 그래도 당당하게 보고한다”의 구조를 유지한다.

좋은 패턴:

```text
✅ 회의록 요약을 완료했습니다.
결론: 314줄의 회의록을 1058줄로 성공적으로 요약했습니다.
```

피해야 할 패턴:

```text
✅ 회의록을 요약했습니다.
결론: 회의는 회의를 낳습니다.
```

후자는 더비가 아니라 사람이 만든 직장인 농담처럼 보인다.
더비는 풍자하지 않는다. 그냥 이상하게 일한다.
