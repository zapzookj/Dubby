# 더비 홈 상태 문구 템플릿 v1

목적: 홈 화면의 `상태`(statusLine) + `현재 업무`(currentWork) 문구. (userId, 날짜) 시드로 하루 1개 고정 선택되며, 노출 이력/쿨다운을 사용하지 않는다.

톤 기준: 푸시와 동일 — 더비는 웃기려고 하지 않는다. 멍청한데 당당하다.

## 후속 확장 규칙

새 항목 추가 시 푸시 문서와 동일한 5문항 기준을 통과해야 한다.
상태는 12자 내외, 업무는 14자 내외로 짧게 유지한다 (홈 카드 잘림 방지 — 시드 파서가 길이 검증).
