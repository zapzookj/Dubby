## 1. 수집하는 정보

더비는 회원가입 없이 동작하는 익명 서비스입니다. 다음 정보를 수집합니다.

| 항목 | 목적 | 보관 기간 |
|---|---|---|
| 기기 식별자(앱 설치 시 생성되는 무작위 ID) | 익명 계정 식별 | 계정 삭제 시까지 |
| 시간대, 언어, 앱 버전, 플랫폼(iOS/Android) | 서비스 제공(일일 리셋, 현지화) | 계정 삭제 시까지 |
| 채팅 내용 | AI 응답 생성, 대화 이력 표시 | 90일 (저장한 답변 제외), 계정 삭제 시 즉시 |
| 더비의 일기장 항목 | 서비스 기능 | 사용자가 삭제할 때까지, 계정 삭제 시 즉시 |
| 푸시 알림 토큰 | 알림 발송 | 알림 해제 또는 계정 삭제 시까지 |
| 구매/구독 상태 (RevenueCat 경유) | 유료 기능 제공 | 관련 법령 보존 기간 |
| 서비스 이용 기록(기능 사용 횟수 등) | 서비스 개선 | 계정 삭제 시까지 |

수집하지 않는 것: 이름, 전화번호, 이메일, 정밀 위치, 연락처.

## 2. 제3자 제공

- **채팅 내용은 AI 응답 생성을 위해 LLM 제공사(OpenRouter 및 그 하위 모델 제공사)에 전송됩니다.** 응답 생성 목적으로만 사용됩니다.
- 결제 처리는 Apple App Store / Google Play 및 RevenueCat을 통해 이루어집니다. 앱은 카드 정보를 수집하지 않습니다.
- 푸시 발송은 Expo Push Service를 경유합니다.

## 3. 민감정보 보호

더비의 일기장은 건강, 성적 지향, 정치 성향, 종교, 정확한 위치 등 민감 정보를 저장하지 않도록
이중 필터(서버 필터 + AI 지시)를 적용합니다. 자해 등 위기 신호가 감지되면 AI 응답 대신
전문기관 안내를 표시하며, 해당 대화는 이후 AI 문맥에서 제외됩니다.

## 4. 삭제 권리

- 일기 항목: 앱에서 개별/전체 삭제 → **즉시 영구 삭제**됩니다.
- 계정: 설정 > 계정 및 데이터 삭제 → 모든 데이터(대화, 일기, 업무 기록, 알림 설정)가
  **즉시 영구 삭제**됩니다. 결제 감사 기록(익명 식별자만 포함)은 분쟁 대응을 위해 보존될 수 있습니다.
